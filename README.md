# Hệ thống thông báo Deadline qua Chat bot Telegram

### Hướng dẫn sử dụng:

1. Đăng ký tài khoản telegram (Mobile hoặc PC)
2. Click đường dẫn [https://t.me/VL_Deadline_bot](https://t.me/VL_Deadline_bot)  để truy cập ChatBot VL_Deadline_bot.
3. Gửi tin nhắn đầu tiên với `*/start*`. 
4. Khi ChatBot yêu cầu đường dẫn **calendars trên trang LMS thì:**
    1. Truy cập Lịch ở LMS
    2. Chọn Quản lý theo dõi
    3. Xuất lịch biểu → Lấy địa chỉ mạng của Lịch (Tùy từng trường sẽ có cách khác nhau)
    
    <img width="627" height="561" alt="image" src="https://github.com/user-attachments/assets/a4468a6c-5e7a-40db-b4c7-9e730bb8fc16" />

    
5. Sau khi lấy được đường link calendars ở LMS thì vào ChatBot → dán link → ấn gửi.
6. Khi hệ thống hiển thị thông báo như hình bên dưới thì đã thành công.
